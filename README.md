# Sistema de Gestión e Información - Acueducto Rural 💧

Este es un sistema web integral diseñado para asociaciones de acueductos rurales. El objetivo es facilitar la transparencia administrativa y mejorar la comunicación con la comunidad mediante una plataforma moderna, rápida y adaptable a dispositivos móviles.

## 🚀 Características Principal
- **Portal Informativo:** Secciones de Inicio, Nosotros y Servicios para la comunidad.
- **Panel de Transparencia:** Gestión y publicación de documentos oficiales (Actas, Tarifas, Informes) en formato PDF.
- **Módulo de Contacto:** Formulario dinámico para reportar daños, PQRS y solicitudes generales con almacenamiento en base de datos.
- **Diseño Responsive:** Optimizado para ser consultado desde teléfonos celulares en zonas rurales.
- **Administración Sencilla:** Interfaz amigable para usuarios con conocimientos básicos de computación.

## 🛠️ Tecnologías Utilizadas
Este proyecto aplica estándares modernos de desarrollo de software:
- **Backend:** Python 3.x con [Flask](https://flask.palletsprojects.com/) (Micro-framework).
- **Frontend:** HTML5, JavaScript y [Tailwind CSS](https://tailwindcss.com/) para el diseño estilizado.
- **Base de Datos:** SQLite3 (Motor ligero y eficiente que no requiere servidores externos costosos).
- **ORM:** SQLAlchemy para la gestión segura de datos.

## 📋 Requisitos Previos
